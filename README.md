Understanding PBWT
----------------------------------------------
While the PBWT algorithm is exceptionally powerful, it can be challenging for newcomers to grasp. To bridge the gap between theoretical papers and production-level code, I have created a series of blog posts (located in the `blog/` directory) designed to provide:

*   **Mental Models:** Intuitive explanations of how the prefix and divergence arrays transform genomic data.
*   **Visual Workflows:** Illustrative diagrams tracing the step-by-step logic of each algorithm.
*   **Simplified Pseudo-code:** Core algorithmic logic extracted and simplified from this source code for better readability.

You can find these resources in `blog/posts/`, starting with the [Introduction](./blog/posts/introduction.qmd).

Feel free to fork repo and modify, if you see anything has to change.
